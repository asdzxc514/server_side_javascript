# Arrays
```js
var _ = require('underscore');  // underscore 모듈을 사용할거다~
var arr = [3, 6, 9, 1, 12];
```
  
`first` : 배열의 첫 번째 요소를 반환 `_.first(arr)`  
`last` : 배열의 마지막 요소를 반환 `_.last(arr)`  
`initial` : 배열의 마지막 항목을 제외한 모든 항목을 반환  `_.initial(arr)`
